This project demonstrates how to transform raw HR data into actionable insights using a complete data engineering and analytics pipeline.

Overview

This project is based on the IBM HR Analytics Employee Attrition and Performance Dataset from Kaggle.
The goal was to design and implement a data pipeline and dashboard that uncover insights on:

Employee attrition and performance trends

Career stagnation and promotion timing

Manager stability and leadership impact

Departmental performance comparisons

Diversity and inclusion metrics

Tools & Technologies

Database:	Microsoft SQL Server
ETL / Data Integration:	SQL Server Integration Services (SSIS)
Data Visualization:	Power BI
Query Language:	SQL / DAX
Data Modeling:	Star Schema + Fact Constellation
